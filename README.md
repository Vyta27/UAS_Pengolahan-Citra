# UAS_Pengolahan-Citra 

Aplikasi pengolahan citra digital berbasis Python dengan GUI Tkinter.

## Fitur
- Konversi (Grayscale, Binary)
- Perbaikan Kualitas (Histogram Eq, Brightness, Sharpening)
- Filtering (Mean, Median, Gaussian)
- Deteksi Tepi (Sobel, Canny, Prewitt)
- Segmentasi (Otsu, K-Means)
- GUI Tkinter + Batch Processing

## Cara Menjalankan
pip install opencv-python pillow numpy matplotlib
python pengolahan_citra.py

## Dataset
21 gambar buah-buahan (folder /dataset)
